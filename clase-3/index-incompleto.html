Plantillas vs Patrones

Entendiendo las Diferencias

Plantillas (Templates) Patrones (Patterns)
🏗️ Estructura completa de página 🧩 Secciones reutilizables
📄 Una por tipo de contenido ✨ Múltiples por página
🔧 Lógica de WordPress 🎨 Solo diseño visual
⚙️ Configuración del tema 📚 Biblioteca reutilizable

Troubleshooting FSE

Problemas Comunes

❌

Problema: Site Editor no disponible

Solución: Verificar tema compatible con FSE

🎨

Problema: Cambios no se reflejan

Solución: Limpiar caché, verificar plantilla activa

📱

Problema: Diseño roto en móvil

Solución: Revisar configuraciones responsive por bloque

⚡

Problema: Sitio lento con FSE

Solución: Optimizar imágenes, minimizar bloques complejos
