Mejores temas FSE gratuitos de WordPress en agosto 2025

  • Ollie
    Minimalista, limpio y con layouts especialmente pensados para startups y landing pages. Gran calidad de patrones predefinidos
  • Neve FSE
    Carga muy rápida, y con plantillas FSE para ecommerce, portfolios y blogs
  • Spectra One
    Tema FSE creado por los desarrolladores del plugin Spectra. Muy enfocado en performance y experiencia de edición visual
  • Frost
    Diseño elegante y modular, ideal para estudios creativos, portfolios o publicaciones profesionales. Incluye numerosos bloques personalizados
  • Bricksy (octubre 2024)
    Un tema de bloques gratuito con un diseño moderno y elegante. Destaca por venir con diseños de patrones pre-diseñados para diferentes nichos de blog como viajes, comida, moda y bodas, lo que lo hace muy versátil.
  • Raft
    Ideal para páginas de aterrizaje, negocios modernos. Plantillas listas para usar, estilos globales, personalización de encabezado, patrones de bloques
  • Gutenify (ver los temas de menos de un año)
    Colección de más de 40 temas de bloques diferentes, todos ellos basados en el plugin Gutenify, que amplía el editor de bloques con nuevas funciones.
  • Greenshift
    Tema FSE (Full Site Editing) que sirve como base para tu sitio. El plugin de bloques Greenshift, un complemento que añade bloques avanzados, animaciones y opciones de diseño al editor de WordPress.
  • Riverbank
    Sencillo tema de WordPress construido para el editor de sitio completo. Tiene una disposición minimalista que se puede utilizar para crear un sitio web de alimentos orgánicos, plantas, bienestar de los animales, y organizaciones sin fines de lucro.
  • Fork
    Tema minimalista para pequeñas empresas, startups, agencias digitales, tiendas de comercio electrónico y sitios web de porfolio. Por defecto, tiene un fondo blanco, pero puedes usar sus estilos globales para añadir una combinación de colores fácilmente.
  • Variations
    Múltiples plantillas y patrones. Enfoque minimalista elegante. Totalmente basado en bloques. Personalización sin límites
  • Extendable
    Tema versátil con 23 variaciones de estilo predefinidas instantáneas. Cambio de diseño con un clic. Compatible con WooCommerce. Soporte multiidioma completo. Optimizado para rendimiento
  • YITH Wonder
    Tema de bloques específico para WooCommerce de la empresa que está detrás de una serie de populares plugins para WooCommerce. Si estás buscando concretamente un tema de bloques para tu tienda WooCommerce, ésta es probablemente una de las mejores opciones a tener en cuenta.
  • Kemet
    Rápido y fiable. Diseñado específicamente para blogs personales, sitios de porfolio y pequeñas tiendas online.
  • Axton
    Tema minimalista para portafolios con variaciones de estilo claro y oscuro. Cuenta con una hermosa tipografía que aprovecha al máximo las imágenes destacadas de las publicaciones. Las líneas contrastantes que separan las diferentes partes del sitio web se ven sobrias y elegantes.
  • ArtsyLens
    Tema artístico con un diseño elegante y una funcionalidad increíble. Es perfecto para artistas, ilustradores, diseñadores, freelancers, fotógrafos y otros profesionales creativos
